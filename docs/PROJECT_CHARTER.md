Sovereign Canon Builder (SCB)

Mission

SCB is a deterministic compiler for structured worldbuilding corpora.

The compiler transforms human-authored lore into a validated,
indexed, reproducible knowledge database suitable for AI retrieval,
analysis, export and tooling.

Primary goals

• Deterministic compilation
• Immutable domain model
• Complete source traceability
• Exhaustive diagnostics
• Reproducible builds
• Stable public API

Non-goals

• AI generation
• Automatic lore modification
• Editing source files
• Guessing missing information
